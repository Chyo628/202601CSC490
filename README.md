# Monitoring and Profiling of AI Agents

Korean version: [README.ko.md](README.ko.md)

This repository hosts the public manuscript snapshot and supporting research artifacts for the paper *Monitoring and Profiling of AI Agents*. It is organized as a public-facing research repository rather than a software package.

## Repository Layout

- [`main.pdf`](main.pdf): Public PDF snapshot of the manuscript.
- [`Citations/`](Citations): Citation and reference materials used to ground claims and review sources.
- [`Citations/Ref PDF/`](Citations/Ref%20PDF): Supplemental local copies of selected reference PDFs and related artifacts.
- [`Data/`](Data): Supporting captured datasets and redacted research artifacts from scoped observation runs.

## Notes

- This public repository does not include the LaTeX source manuscript or bibliography database.
- The tracked files are limited to the public PDF snapshot, citation/reference materials, and supporting data artifacts.
- This README is intentionally focused on public repository orientation rather than internal editing workflow.
